# Project-grafa
Project of subject ZPG (Základy počítačové grafiky) - OpenGL library. 

**Project contains:** 
- Transformation (rotation, scale, translation)
- Default, Lambert, Phong shaders
- Type of lights (point, direct, spot lights)
- Assimp library
- Skybox
- ID of object
- Bezier curve
- etc.
